# Contributions adder.

This is supposed to add contributions to spice up your repo
For terminal you can use **powershell**, **cmd**, **git bash** or your prefered terminal

---
## Procedure

### Step 1: Make a **private** repo:
Navigate to the repository section and add a private repo of your choice
>[!TIP]
>Adding the repo as a **PRIVATE** repo prevents people from knowing that this is going on
>In short **MIND YOUR SECRECY**

### Step 2: Ensure python is installed
Verify the installation by running the following in your terminal
```
python --version
```
### Step 3: Clone this repo:
Run the command in the terminal
```
git clone git@github.com:data-spice/add_cont.git
```
### Step 4: Move to the add_cont directory
```
cd add_cont
```
### Step 5: Run the python file:
> [!CAUTION]
> After running the python file add the number of contributions you want to add then continously press **enter** without typing anything in the proceeding fields.

>[!TIP]
> Enter a logical number of commits in accordance to you repo to preserve authenticity.

#### Use the command below to run the file:

```
python main.py
```
